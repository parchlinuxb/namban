# NAMBAN

A simple gui tool for set dns settings based on GTK4 and Adwaita, related to [parchlinux project](https://github.com/parchlinux/)

![namban img 1](/img/1.png)

![namban img 2](/img/2.png)

## develop environment

1. python > 3.10
2. Gtk > 4.10
3. Adw
4. PyGObject
5. A lot of milk for mental peace
